# 🧑‍💼 Employee Management System

A full-stack web application to manage employees, tasks, and roles.

---

## 🚀 Live Demo

https://employee-management-system-xyz.vercel.app/login

---

## 🛠️ Tech Stack

### 🎨 Frontend

* ⚛️ React (Vite)
* 🎨 Tailwind CSS

### 🔧 Backend

* 🟢 Node.js
* 🚀 Express.js

### 🗄️ Database

* 🍃 MongoDB

---

## ✨ Features

* 🔐 Authentication & Authorization (Admin / Manager / User)
* 👨‍💼 Employee Management (Create, Update, Delete, View)
* 📋 Task Management System
* 📊 Dashboard (statistics & charts)
* 🔍 Filtering & Searching
* 🌙 Fully responsive UI

## ⚙️ Installation & Setup

### 📥 Clone the repo

```id="n2k8xz"
git clone https://github.com/EngineerMohamed2016/employee-management-system
```

### 💻 Frontend Setup

```id="d7r3qp"
cd front
npm install
npm run dev
```

### 🖥️ Backend Setup

```id="x4mt1v"
cd back
npm install
npm run dev
```
