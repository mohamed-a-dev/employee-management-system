# 🧑‍💼 Employee Management System

A full-stack web application to manage employees, tasks, and roles داخل الشركة.

---

## 🚀 Live Demo

https://private-employee-management-system-five.vercel.app/

---

## 🛠️ Tech Stack

### Frontend

* React (Vite)
* Tailwind CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

---

## ✨ Features

* 🔐 Authentication & Authorization (Admin / Manager / User)
* 👨‍💼 Employee Management (Create, Update, Delete, View)
* 📋 Task Management System
* 📊 Dashboard (statistics & charts)
* 🔍 Filtering & Searching
* 🌙 Responsive UI

## ⚙️ Installation & Setup

### 1. Clone the repo

https://github.com/EngineerMohamed2016/employee-management-system

### 2. Frontend Setup

cd front
npm install
npm run dev

### 3. Backend Setup

cd back
npm install
npm run dev

---

## 🔑 Environment Variables

### Backend (.env)

PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret

### Frontend (.env)

VITE_API_URL=http://localhost:5000
